# Getting started

## Installation

Install `@ogs-gmbh/stdx` using your preferred package manager:

::: code-group
```bash [npm]
npm install @ogs-gmbh/stdx
```
```bash [yarn]
yarn add @ogs-gmbh/stdx
```
```bash [pnpm]
pnpm install @ogs-gmbh/stdx
```
:::
